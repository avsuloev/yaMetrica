# Yandex Metrica for Symfony (playground)

Learning to work with Yandex Metrika API before implaementing in actual projects.

## Installation

Install Symfony and Composer packages on your OS, then run

```shell
composer install --optimize-autoloader
symfony server: --port=8080
```
or use PHP built-in server
```shell
php -S localhost:8080 -t public/
```
